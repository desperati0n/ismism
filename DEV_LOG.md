│   ├── ResultCard.tsx

**根目录**的 `App.tsx`, `main.tsx`, `components/`, `styles/` 是实际运行的代码。
**src/** 目录是历史遗留，**不会被加载**！

## 📦 组件说明

### App.tsx - 主应用
**功能**: 整体布局和状态管理

**状态**:
- `sliderValues`: 四个维度的当前值 `['1','2','3','4']`
- `searchResults`: 搜索结果列表
- `hasSearched`: 是否已搜索
- `selectedIsm`: 当前查看的主义详情

**布局**:
```
┌─────────────────────────────────────┐
│  WaveTextBackground (波浪文字背景)   │
│  ┌─────────────────────────────┐   │
│  │   TopNavigation (导航栏)     │   │
│  └─────────────────────────────┘   │
│  ┌─────────────────────────────┐   │
│  │      白色居中卡片             │   │
│  │  ┌───┬───┬───┬───┬───┐      │   │
│  │  │ 1 │ 2 │ 3 │ 4 │ 🔍│      │   │ ← GridSliders + SearchButton
│  │  └───┴───┴───┴───┴───┘      │   │
│  │  ─────────────────────      │   │ ← 分隔线
│  │  搜索结果 / 详情 / 说明      │   │
│  └─────────────────────────────┘   │
│  ● (粉红装饰点)                     │
└─────────────────────────────────────┘
```

---

### WaveTextBackground.tsx - 波浪文字背景
**功能**: 创建多条倾斜滚动的"主义"文字条带

**原理**:
- 7条文字带，不同角度(-3° 到 -12°)
- 使用 CSS `@keyframes scrollText` 实现无限滚动
- 交替正向/反向滚动

---

### TopNavigation.tsx - 顶部导航
**功能**: 显示标题和菜单按钮

**样式**: 
- 透明毛玻璃背景
- 左侧"主义主义"标题
- 右侧黑色圆形菜单按钮

---

### GridSlider.tsx - 维度选择器
**功能**: 点击切换 1→2→3→4→$→1 循环

**特性**:
- 60x60px 方形按钮
- 黑色边框，白色背景
- $ 符号显示为粉红色

---

### SearchButton.tsx - 搜索按钮
**功能**: 触发搜索

**样式**:
- 60x60px 纯黑方块
- 白色搜索图标

---

### ResultCard.tsx - 结果卡片
**功能**: 显示搜索结果列表项

**样式**:
- 底部黑线分隔
- 悬停时浅粉色背景

---

### IsmDetail.tsx - 详情页
**功能**: 显示主义的完整信息

**内容**:
- 返回按钮
- 编码标签（黑底白字）
- 名称（粗体大字）
- 描述文本
- 维度解析（4个色块）

---

## 🎭 styles/globals.css - 全局样式

### 动画
```css
/* 淡入上升 */
.animate-fadeInUp

/* 文字滚动 */
.animate-scroll        /* 正向 */
.animate-scroll-reverse /* 反向 */
```

### 自定义类
```css
.dot-pink       /* 粉红装饰圆点 24px */
.dot-black      /* 黑色圆形按钮 40px */
.result-item    /* 结果列表项样式 */
.text-ribbon    /* 文字条带容器 */
```
